## ğŸ“¦ Component Breakdown

### 1. Frontend Layer (`frontend/`)
- **Technology**: Next.js 16, React, TypeScript, Tailwind CSS
- **Key Files**:
  - `app/page.tsx` - Main dashboard UI
  - `app/layout.tsx` - Root layout wrapper
  - `app/globals.css` - Global styles
- **Responsibilities**:
  - User input collection (profile selection or custom JSON)
  - API communication via fetch
  - Risk visualization (color-coded cards, progress bars)
  - Error handling and loading states

### 2. API Layer (`api/`)
- **Technology**: FastAPI, Pydantic, Uvicorn
- **Key Files**:
  - `api/main.py` - REST API endpoints
- **Endpoints**:
  - `POST /score` - Score custom borrower data
  - `POST /predict` - Score predefined profiles
  - `GET /health` - Health check
- **Responsibilities**:
  - Request validation
  - CORS handling
  - Model lazy loading
  - Exception handling

### 3. Business Logic Layer (`model/`)
- **Key Files**:
  - `model/risk_scoring.py` - Main scoring orchestrator
  - `model/feature_engineering.py` - Feature extraction
  - `model/config.py` - Profile configurations
- **Responsibilities**:
  - Feature engineering (9 behavioral features)
  - Risk classification logic
  - Recommendation generation
  - Response formatting

### 4. ML Inference Layer (`model/`)
- **Key Files**:
  - `model/risk_model.pkl` - Serialized trained model
- **Model Architecture**:
  - Base: RandomForestClassifier (50 estimators)
  - Calibration: Isotonic regression
  - Input: 9 features
  - Output: 3-class probabilities
- **Responsibilities**:
  - Probability prediction
  - Risk class assignment

### 5. Training Pipeline (`model/`)
- **Key Files**:
  - `model/train_model.py` - Training script
  - `data/simulate_data.py` - Synthetic data generator
- **Responsibilities**:
  - Generate 600 synthetic samples
  - Train Random Forest
  - Apply calibration
  - Serialize model

### 6. CLI Interface (`app/`)
- **Key Files**:
  - `app/cli.py` - Terminal interface
- **Responsibilities**:
  - Interactive profile selection
  - Direct model invocation
  - Console-based risk display
